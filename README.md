# Text Translator

Text-Translator is an app that translates simple words from one language to another

## Building and running on localhost

- Clone this repo

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
npm run dev
```
